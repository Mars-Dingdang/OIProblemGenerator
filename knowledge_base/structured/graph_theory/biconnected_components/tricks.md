- **边双连通分量具有传递性，点双不具有**：这意味着边双可以像强连通一样用栈收缩，而点双需要特殊处理根节点和割点。
- **Tarjan 求边双等价于求 DFS 树上的强连通分量**：因为非树边会形成环，覆盖的边都不是桥；当 `dfn[u] == low[u]` 时，栈中从 `u` 到顶的节点构成一个边双。
- **差分优化判桥**：对每条非树边 `(u,v)`，在深度小的端点 `-1`，深度大的端点 `+1`，子树差分和为 0 的树边即为桥。
- **点双连通分量输出时机**：在回溯时判断 `low[v] >= dfn[u]`，说明 `u` 是割点，将栈中从 `v` 到当前顶的元素弹出作为一个点双，并加入 `u`。
- **孤立点处理**：若根节点无邻接边，应单独作为一个点双。
- **重边不影响双连通性**：多条边连接两点不会产生桥，需注意建图时避免重复计数但保留多重边语义。
- **内存优化技巧**：在空间受限场景下（如 CEOI 管道题），可用并查集合并被完全覆盖的非树边，减少冗余信息存储。
- **新图建模法（点双）**：将树边映射为新图中的点，非树边路径连接对应的新点，新图中每个连通块对应原图的一个点双连通结构。
