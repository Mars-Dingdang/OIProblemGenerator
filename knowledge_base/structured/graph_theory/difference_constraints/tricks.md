- **变量平移不变性**：如果 \{x_1, x_2, ..., x_n\} 是一组解，那么 \{x_1 + d, x_2 + d, ..., x_n + d\} 也是一组解。因此可以添加超级源点 $0$ 并连边 $0 \to i$ 权值为 $0$，以保证图连通并正确初始化。

- **最长路 vs 最短路**：
  - 当约束是 $x_i - x_j \leq c$，建边 $j \to i$ 权 $c$，求最短路（存在负环则无解）。
  - 当约束是 $x_i - x_j \geq c$，即 $x_j - x_i \leq -c$，建边 $i \to j$ 权 $-c$，但更直观的是转为最长路模型：$x_i \geq x_j + c$，建边 $j \to i$ 权 $c$，跑最长路，出现正环则无解。

- **常用变形**：
  - $x_a = x_b$ 拆成两个不等式：$x_a - x_b \leq 0$ 和 $x_b - x_a \leq 0$。
  - $x_a - x_b \geq c \Rightarrow x_b - x_a \leq -c$，反向建边权 $-c$。

- **对数变换技巧**：对于形如 $\frac{x_i}{x_j} \leq c$ 的约束，两边取对数得 $\log x_i - \log x_j \leq \log c$，转化为标准差分约束问题。

- **建模方向统一建议**：推荐统一使用最长路模型处理 $\geq$ 类型约束（常见于最大化问题），避免符号混乱。

- **判环方式选择**：
  - 使用 SPFA 判环时，记录入队次数 `cnt[i]`，若某点入队 $\geq n$ 次，则存在负环（或正环，取决于模型）。
  - Bellman-Ford 松弛 $n-1$ 次后仍能更新，则存在负环。

- **初始化技巧**：从虚拟源点 $0$ 向所有点连 $0$ 边，确保所有变量可达，且初始解合法（如全 $0$ 解满足 $x_i \leq x_0 + 0$）。

- **稀疏图适用**：SPFA 在随机稀疏图上表现良好，但最坏情况 $O(nm)$，注意题目是否卡 SPFA。
