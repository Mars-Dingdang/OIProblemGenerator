- **欧拉公式**：对于连通平面图，|V| − |E| + |F| = 2。推广到k个连通分支时为 |V| − |E| + |F| = k + 1。
- **边数上界**：简单可平面图满足 |E| ≤ 3|V| − 6，说明其为稀疏图，可用于复杂度分析或剪枝。
- **面次数和**：所有面的次数之和等于 2|E|，计算时割边贡献两次。
- **对偶图技巧**：平面图的最小割问题可转化为对偶图上的最短路问题。通过构造对偶图并将原图边权赋给对偶边，s-t 割对应 f⁺ 到 f⁻ 的路径。
- **极大平面图特征**：当 |V| ≥ 3 时，极大平面图每个面都是三角形（次数为3），且满足 |E| = 3|V| − 6，|F| = 2|V| − 4。
- **外平面图性质**：极大外平面图有 |E| = 2|V| − 3，内部面数为 |V| − 2，外部面是长度为 |V| 的环，所有内部面为三角形。
- **平面性判定**：使用 Kuratowski 定理（不含与 K₅ 或 K₃,₃ 同胚的子图）或 Wagner 定理（不能收缩出 K₅ 或 K₃,₃）进行理论判断；实际算法可用 Boyer–Myrvold 算法（线性时间，输出嵌入或禁用子图）。
- **构造性证明技巧**：如极大外平面图必存在度数为2的顶点，常用于归纳法证明。
- **四色定理应用**：任何无自环平面图是4-可着色的，可用于染色类问题简化。
- **Fáry定理**：简单可平面图存在全由直线段构成的平面嵌入，几何题中可假设直线绘制。
- **注意对偶图依赖嵌入方式**：同一可平面图的不同嵌入可能产生不同构的对偶图，需固定嵌入后再建对偶图。
