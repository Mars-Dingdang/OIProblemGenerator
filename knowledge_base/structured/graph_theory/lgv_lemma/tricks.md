- **关键观察**：LGV 引理通过构造行列式来自动处理带符号的不相交路径组求和。所有相交路径组在行列式展开中成对抵消，仅留下不相交路径组的贡献。

- **路径权值设计**：$\omega(P)$ 可以是边权乘积，也可扩展为生成函数，适用于带权路径计数问题。

- **符号简化**：当起点和终点自然有序（如 $a_1 < a_2 < \cdots < a_n$, $b_1 < b_2 < \cdots < b_n$），且图结构保证唯一可能的不相交匹配是 $a_i \to b_i$ 时，排列 $\sigma$ 必为恒等排列，此时 $(-1)^{t(\sigma)} = 1$，可直接计算行列式而不担心符号问题（如 HDU 5852）。

- **网格图路径数**：在只能向右或向下移动的网格中，从 $(x_1,y_1)$ 到 $(x_2,y_2)$ 的路径数为组合数 $\binom{(x_2-x_1)+(y_2-y_1)}{x_2-x_1}$，常用于快速计算 $e(u,v)$。

- **障碍处理**：若存在障碍点，可通过动态规划预处理 $f(u,v)$ 路径数（如 CF348D），再代入矩阵。

- **复杂度优化**：矩阵大小通常为 $k \times k$（$k$ 为路径数），即使图很大，只要 $k$ 小（如 $k \leq 100$），高斯消元 $O(k^3)$ 是可接受的。

- **问题转化**：将“不相交路径计数”转化为“带符号路径组求和”，利用代数结构（行列式）自动完成容斥，避免显式容斥原理编程。
