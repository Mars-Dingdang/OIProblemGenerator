- **Dist 定义注意**：代码中通常将空节点的 `dist` 设为 `-1`，这样非空节点的 `d = rs.d + 1` 可直接更新，无需特判。
- **无需交换写法**：通过 `rs(x)` 动态取 `dist` 较小的子节点作为右儿子，避免显式交换左右儿子，简化逻辑。
- **懒标记处理**：在根节点打标记，`merge` 或访问子节点前调用 `pushdown` 下传。适用于加、乘等不改变堆序的操作。
- **删除任意节点**：合并该节点的两个子树，然后从父节点向上 `pushup` 更新 `dist` 和左偏性，递归终止条件是 `dist` 不再变化。
- **并查集维护堆顶**：由于左偏树深度可能为 $O(n)$，查找所在堆的根应使用并查集路径压缩，不能暴力跳父亲。
- **启发式标记下传**：合并时将较小堆的标记下传至所有节点，较大堆的标记保留为新堆标记，保证每个节点最多被下传 $O(\log n)$ 次，总复杂度 $O(n \log n)$。
- **树上应用模式**：类似线段树合并，在 DFS 回溯时合并子树对应的堆，按题意弹出元素或更新答案，如「城池攻占」。
- **全局最大值维护**：用 `multiset` 维护所有堆顶值，支持快速查询全局最值，配合懒标记实现高效更新。
