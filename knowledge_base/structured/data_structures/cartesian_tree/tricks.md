- **单调栈构建**：按 $k$ 升序插入，维护右链（从根向右儿子走到叶的路径），使用单调栈维护当前右链节点，每个节点最多进出一次，时间复杂度 $O(n)$。
- **唯一性**：若所有 $k$ 和 $w$ 均互不相同，则笛卡尔树结构唯一。
- **子树连续性**：以数组下标为 $k$ 时，任意子树对应的下标构成连续区间，便于区间问题转化。
- **与 Treap 的关系**：Treap 是笛卡尔树的特例，其中 $w$ 随机赋值；若提前对 $k$ 排序，可用单调栈线性构建 Treap。
- **经典应用**：在最大子矩形问题中，以 $h_i$ 为堆权值构建小根堆型笛卡尔树，枚举每个节点作为矩形高度，利用子树连续性和堆性质计算最大面积，总复杂度 $O(n)$。
- **DFS 合并信息**：常配合 DFS 在树上进行 DP 或信息聚合，利用结构特性避免显式 RMQ。
