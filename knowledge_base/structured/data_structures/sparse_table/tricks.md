- **利用 `__builtin_clz` 或预处理 `Log` 数组**：避免每次调用 `log` 函数，提升查询效率。
- **优先将 `log n` 维度作为第一维**：提高缓存局部性，优化常数表现。
- **覆盖区间使用两个重叠块**：利用可重复贡献性质，将任意区间 `[l, r]` 拆分为 `[l, l+2^s-1]` 和 `[r-2^s+1, r]`，其中 `s = floor(log2(r-l+1))`，保证正确性的同时实现 O(1) 查询。
- **适用于满足结合律和幂等性的运算**：如 `max`, `min`, `gcd`, `&`, `|` 等；不适用于 `sum` 等不可重复贡献的运算。
- **ST 表维护 GCD 的复杂度分析**：预处理时间复杂度为 Θ(n(log n + log w))，单次查询为 Θ(log w)，虽查询不如线段树优，但代码简洁、易于实现。
- **势能分析法证明复杂度**：在 GCD 预处理中，每次非终递归至少使某数减半，总递归次数被势能函数均摊为 O(n log w)，结合循环部分得总复杂度。
- **输入输出优化建议**：面对大量查询时，启用 `ios::sync_with_stdio(false)` 和 `cin.tie(nullptr)`。
