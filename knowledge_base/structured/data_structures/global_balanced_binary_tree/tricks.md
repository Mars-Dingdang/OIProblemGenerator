- **加权中点建树**：在构建每条重链对应的二叉树时，使用轻儿子子树大小之和（+1）作为权重求中点，保证树高为 $ O(\log n) $。
- **跳转路径分解**：从任意节点到根的路径由 $ O(\log n) $ 条轻边和重边组成。跳轻边对应切换重链，跳重边在二叉树内进行。
- **标记永久化**：避免 `pushdown` 操作，通过在查询/更新时累计标记影响来减少常数，特别适合只修改整条链或子树的场景。
- **子树信息维护技巧**：`ss[u]` 表示二叉树中以 `u` 为根的重子树节点数（原树中对应重链片段长度），用于快速计算贡献。
- **轻边‘认父不认子’**：轻边仅记录父指针，不能反向访问，确保结构清晰且防止循环引用。
- **与动态 DP 结合**：可用于维护树上动态规划的状态转移矩阵，将树剖分与矩阵乘法结合实现高效的动态更新（如最大独立集问题）。
