## 关键观察
1. **倍增结构**：牛顿迭代的核心是倍增。从模 x^1 的解开始，每次将精度加倍（x^n → x^{2n}），利用泰勒展开的高阶项在模意义下为零的性质简化计算。
2. **泰勒展开截断**：因为 f(x) - f_{n/2}(x) 的最低非零项次数至少为 ⌈n/2⌉，所以展开式中 i≥2 的项在模 x^n 下为零，只需保留前两项。
3. **初始条件**：需要存在 f1 使得 G(0, f1)=0 且 ∂G/∂y(0, f1)≠0。这保证了偏导数可逆，迭代可进行。

## 优化技巧
1. **利用已有多项式操作**：牛顿迭代常依赖多项式乘法、求逆等基本操作，可用 NTT 优化至 O(n log n)。
2. **迭代公式统一**：不同问题的迭代公式形式类似：f_{2n} = f_n - G(f_n) / G'(f_n)。只需根据具体 G 实现对应的函数。
3. **避免重复计算**：在实现中，可缓存中间结果（如 f_n 的逆）以减少计算量。

## 常见问题转化
1. **多项式求逆**：设 G(f) = 1/f - h，则迭代公式为 f_{2n} = 2f_n - f_n^2 h。
2. **多项式开方**：设 G(f) = f^2 - h，则迭代公式为 f_{2n} = (f_n^2 + h) / (2f_n)。
3. **多项式指数函数**：设 G(f) = ln f - h，则迭代公式为 f_{2n} = f_n (1 - ln f_n + h)。
4. **整数模素数幂平方根**：算法可迁移到整数模 p^n 情形，公式相同，只需在整数模意义下计算。

## 注意事项
1. **常数项要求**：多项式开方要求 h(0)≠0（有常数项），否则初始解可能不存在。
2. **唯一性**：初始解 f1 的选择可能影响最终解（如开方时有正负两个解）。
3. **收敛性**：在满足初始条件下，牛顿迭代总是收敛到模 x^n 的解。
4. **时间复杂度**：每次迭代需 O(n log n) 时间，总复杂度 T(n) = T(n/2) + O(n log n) = O(n log n)。
