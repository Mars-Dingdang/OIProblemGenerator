## 关键技巧

### 1. 标准形式转换
- **最大化转最小化**：目标函数系数取负
- **不等式转等式**：添加松弛变量或剩余变量
- **自由变量处理**：x = x⁺ - x⁻，其中 x⁺, x⁻ ≥ 0
- **右侧常量非负**：约束两边同时乘以 -1

### 2. 对偶理论应用
- **弱对偶定理**：对偶问题最优值 ≤ 原问题最优值
- **强对偶定理**：若原问题或对偶问题可行，则最优值相等
- **互补松弛条件**：最优解满足 xᵢ*(Aᵀy - c)ᵢ = 0
- **原始-对偶方法**：通过求解对偶问题改进原问题解

### 3. 整数规划特殊情形
- **全幺模矩阵**：若约束矩阵A全幺模且b为整数，则线性规划松弛的整数解自动成立
- **常见全幺模结构**：
  - 网络流问题的节点-弧关联矩阵
  - 二分图匹配的关联矩阵
  - 区间图的关联矩阵

### 4. 竞赛实用技巧
- **图解法**：仅适用于2变量问题，直观找顶点最优解
- **冗余约束识别**：通过求解线性规划判断约束是否冗余
- **可行性检验**：使用Farkas引理判断不等式组是否有解
- **原始-对偶算法**：用于网络流、匹配等问题，如匈牙利算法、最小费用流算法

### 5. 数值稳定性
- **避免退化**：使用Bland规则防止循环（选择最小下标的进基/出基变量）
- **精度处理**：使用EPS比较浮点数，避免除零错误
- **大系数处理**：对系数进行适当缩放提高数值稳定性

### 6. 问题转化模式
- **多商品流**：可转化为线性规划，但网络流方法更高效
- **资源分配**：典型线性规划应用，约束为资源限制
- **生产计划**：目标为利润最大化，约束为生产能力
- **组合优化**：许多NP问题可写为整数线性规划，再松弛求解
