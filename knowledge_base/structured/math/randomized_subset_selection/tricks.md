- **两阶段算法**：先随机生成初始子集，再通过添加/删除调整到目标大小 n/2。
- **Hoeffding 不等式应用**：分析偏移量 |X - E[X]|，其中 X 是初始子集大小。取 t = c√n，可得失败概率 ≤ 2e^{-c^2}，允许 Θ(√n) 偏移以实现高概率保证。
- **期望复杂度**：期望抛硬币次数为 n + 2√(πn)⌈log₂ n⌉，其中 n 来自初始生成，√n log n 来自调整阶段。
- **问题转换**：该算法可用于近似随机抽样，平衡随机性与效率。
- **练习关联**：Balls and Bins 问题中，最多球的盒子以高概率有 Ω(log n / log log n) 个球，可用类似概率分析。
- **优化方向**：调整阶段可优化为批量操作，减少随机数生成开销。
