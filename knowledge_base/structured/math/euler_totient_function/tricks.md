## 关键性质
- **积性函数**: 若 gcd(a,b)=1，则 φ(ab)=φ(a)φ(b)。特别地，当 n 为奇数时 φ(2n)=φ(n)。
- **求和公式**: n = Σ_{d|n} φ(d)。这是欧拉反演的核心。
- **质数幂**: 若 n=p^k，则 φ(n)=p^k - p^{k-1} = p^{k-1}(p-1)。
- **通项公式**: 若 n = Π p_i^{k_i}，则 φ(n) = n * Π (1 - 1/p_i)。

## 常见应用模式
1. **化简 gcd 求和**: 利用 n = Σ_{d|n} φ(d)，可将 Σ_{i=1}^n gcd(i,n) 转化为 Σ_{d|n} ⌊n/d⌋ φ(d)。
2. **数论分块优化**: 对于多组查询，预处理欧拉函数前缀和，结合数论分块可将求和复杂度降至 O(√n)。
3. **线性筛预处理**: 当需要计算 1~n 所有欧拉函数值时，使用线性筛可在 O(n) 内完成。

## 实现技巧
- **单值计算**: 直接质因数分解，每找到一个质因子 p，执行 ans = ans / p * (p-1)。
- **多值计算**: 使用线性筛，利用积性函数性质递推。
- **注意溢出**: 在 ans = ans / i * (i-1) 中先除后乘，避免中间结果溢出。
