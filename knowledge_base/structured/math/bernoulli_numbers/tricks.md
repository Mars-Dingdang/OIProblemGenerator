- **等幂和公式**: S_m(n) = 1/(m+1) * Σ_{k=0}^{m} C(m+1, k) * B_k * n^{m+1-k}。这是伯努利数最经典的应用。
- **递推关系**: Σ_{j=0}^{m} C(m+1, j) * B_j = 0 (m>0)，B_0 = 1。这是定义伯努利数的常用递推式，可用于 O(n^2) 预处理。
- **指数生成函数**: B(z) = z / (e^z - 1)。利用此生成函数，可以通过多项式求逆在 O(n log n) 时间内求出前 n 项伯努利数。
- **奇数项性质**: 对于 n > 1 且 n 为奇数，B_n = 0。因此只需计算偶数项。
- **模运算注意**: 伯努利数是有理数，在模素数下需要预处理逆元来计算分数。递推公式中的除法转化为乘逆元。
- **多项式加速**: 使用 NTT/FFT 进行多项式求逆和乘法，可以将预处理复杂度优化到 O(n log n)。
- **等幂和变体**: 公式适用于求和从 0 到 n-1，若求和从 1 到 n，只需将结果加上 n^m 或调整公式。
- **与其他序列关联**: 伯努利数与欧拉数、黎曼ζ函数（ζ(1-n) = -B_n / n）有深刻联系，可用于数论问题。
