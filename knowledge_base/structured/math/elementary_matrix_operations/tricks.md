## 初等矩阵的核心性质

### 1. 行列式变化规律
- **倍乘矩阵** $D_i(k)$：行列式为 $k$，操作后矩阵行列式变为原来的 $k$ 倍
- **对换矩阵** $P_{ij}$：行列式为 $-1$，操作后矩阵行列式变为原来的 $-1$ 倍
- **倍加矩阵** $T_{ij}(k)$：行列式为 $1$，操作后矩阵行列式不变

### 2. 逆矩阵公式
- $D_i(k)^{-1} = D_i(\frac{1}{k})$（要求 $k \neq 0$）
- $P_{ij}^{-1} = P_{ij}$（对换矩阵的逆就是自身）
- $T_{ij}(k)^{-1} = T_{ij}(-k)$（倍加矩阵的逆只需取反系数）

### 3. 矩阵分解技巧
- **上三角矩阵分解**：若主对角线均为1，可分解为一系列倍加矩阵的乘积
- **对角矩阵分解**：若主对角线元素均非零，可分解为倍乘矩阵的乘积
- **置换矩阵分解**：任何置换矩阵可分解为对换矩阵的乘积

### 4. 计算优化
- **左乘 vs 右乘**：左乘对应行变换，右乘对应列变换（口诀：左行右列）
- **操作顺序**：倍乘和对角阵乘法可交换，但倍加和对换矩阵乘法不可交换
- **高斯消元应用**：只需倍加和倍乘操作即可完成消元，对换操作为辅助

### 5. 秩与可逆性
- 矩阵可逆 ⇔ 可表示为初等矩阵的乘积 ⇔ 秩为n ⇔ 行列式非零
- 初等变换不改变矩阵的秩
- 标准形矩阵中1的个数等于矩阵的秩

### 6. 记忆技巧
- 用 $E_{ij}$ 表示法记忆初等矩阵：
  - $D_i(k) = I_n + (k-1)E_{ii}$
  - $P_{ij} = I_n - E_{ii} - E_{jj} + E_{ij} + E_{ji}$
  - $T_{ij}(k) = I_n + kE_{ij}$

### 7. 竞赛应用
- **快速求逆**：对增广矩阵 $[A|I]$ 进行初等行变换得到 $[I|A^{-1}]$
- **行列式计算**：通过初等变换化为上三角矩阵，对角线乘积即为行列式
- **线性方程组**：通过初等行变换化为行阶梯形求解
- **矩阵对角化**：通过初等行列变换将方阵化为对角阵
