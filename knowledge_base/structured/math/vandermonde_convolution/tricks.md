## 关键观察
- **组合意义**：从大小为 n+m 的集合中选 k 个元素，等价于将集合划分为大小分别为 n 和 m 的两部分，然后从第一部分选 i 个，从第二部分选 k-i 个，对所有 i 求和。
- **生成函数证明**：利用 (1+x)^n(1+x)^m = (1+x)^{n+m} 的二项式展开，比较系数即得公式。

## 常见推论
1. **对称形式**：∑_{i=-r}^{s} C(n, r+i) C(m, s-i) = C(n+m, r+s)
2. **特殊值**：∑_{i=1}^{n} C(n, i) C(n, i-1) = C(2n, n-1)
3. **平方和**：∑_{i=0}^{n} C(n, i)^2 = C(2n, n)

## 应用技巧
- **化简求和**：将双重求和转化为单组合数，降低计算复杂度。
- **组合恒等式证明**：作为中间引理推导其他恒等式。
- **生成函数结合**：与多项式乘法、卷积运算结合使用。
- **参数变形**：通过变量替换将问题转化为范德蒙德卷积形式。

## 注意事项
- 公式要求组合数在定义范围内（即 0 ≤ i ≤ n, 0 ≤ k-i ≤ m）。
- 模数较大时需预处理阶乘和逆元。
- 推论形式多样，需根据问题灵活变形。
