## 高斯消元法的关键技巧

### 1. 精度处理
- 使用 `EPS = 1e-9` 处理浮点数误差
- 比较时使用 `abs(x) < EPS` 而非 `x == 0`
- 选择主元时使用绝对值最大的元素，提高数值稳定性

### 2. 无解与无穷多解判断
- 如果某行系数全为0但常数项不为0 → 无解
- 如果某行系数全为0且常数项为0 → 无穷多解
- 自由变量的个数 = n - rank(A)

### 3. 模意义下的高斯消元
```cpp
// 模质数p下的高斯消元
int inv(int x) { return pow_mod(x, p-2, p); }
// 消元时使用模逆元进行除法
```

### 4. 异或方程组优化
- 使用 `bitset` 存储增广矩阵，时间复杂度 O(n²m/ω)
- 异或运算满足结合律，消元时使用 `^=` 操作
- 无需进行乘除运算，直接异或消元

### 5. 矩阵求逆技巧
- 构造增广矩阵 (A|I)
- 通过行变换化为 (I|A⁻¹)
- 如果左半部分不能化为单位矩阵，则矩阵不可逆

### 6. 行列式计算优化
- 高斯消元过程中记录行交换次数，确定行列式符号
- 消元后行列式 = 对角线元素乘积 × (-1)^交换次数
- 遇到全零列直接返回0

### 7. 稀疏矩阵处理
- 对于稀疏矩阵，可以使用特殊数据结构存储非零元素
- 时间复杂度可优化到 O(nnz)，其中 nnz 是非零元素个数

### 8. 常见问题转化
- 线性基问题可转化为异或方程组
- 图论中的 Kirchhoff 矩阵树定理使用行列式计算
- 概率期望问题中的线性方程组求解

### 9. 数值稳定性改进
- 完全主元消去法：同时考虑行和列的主元选择
- 缩放主元消去法：先对行进行缩放
- 迭代改进法：对解进行迭代修正
